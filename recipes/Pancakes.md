# Pancakes

## Ingridients

| ingridient | amount |
| --------------------- | ----------- |
| Mehl                  | 250g        |
| Eier                  | 2          |
| Salz                  | 1 Prise     |
| Backpulver            | 3 TL        |
| Milch                 | 175 ml |
| Zucker                | 1 EL |

## Instructions

Eier trennen und Eiweiß aufschlagen. Rest vermischen, Eischnee unterheben und anschließend in der Pfanne braten.
